# React Workshop
Learning about the fundamentals of React at RailsConf 2017

# Local Setup

- Clone the repo
- Inside the directory run the following commands
- `bundle install`
- `rake db:create`
- `rake db:migrate`
- `rake db:seed`
- `rails server`


# Technologies

- Rails 5.0
- sqlite3
- react-rails
- materialize-sass
